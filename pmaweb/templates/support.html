Types of questions and where to ask

  • How do I? — Stack Overflow
  • I got this error, why? — Stack Overflow
  • I got this web server error (like "404 - Not Found") — serverfault.com or webmasters.stackexchange.com
  • I got this error on the current stable version and I'm sure it's a bug — Please file an issue
  • I have an idea/request — Please file an issue
  • I have a question about the code itself — Use our developers@phpmyadmin.net mailing list

Mailing Lists

  • For news/announcements: news@phpmyadmin.net
  • For developers: developers@phpmyadmin.net
  • For translators: translators@phpmyadmin.net
  • Git commit notifications: git@phpmyadmin.net
  • Mailing lists overview
